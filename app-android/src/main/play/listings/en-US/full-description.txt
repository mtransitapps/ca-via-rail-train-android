This app adds Canada VIA Rail trains information to MonTransit.

This app provides the planned schedule as well as real time service statuses and news from viarail.ca and @VIA_Rail on Twitter.

VIA Rail trains serve Vancouver, Edmonton, Saskatoon, Winnipeg, Toronto, Ottawa, Montreal, Halifax... in Canada.

Once this application is installed, the MonTransit app will display trains information (schedule...).

This application only has a temporary icon: download the MonTransit app (free) in the "More ..." section bellow or by following this Google Play link https://goo.gl/pCk5mV

You can install this application on the SD card but it is not recommended.

The information comes from the GTFS file provided by VIA Rail under Open Government Licence - Canada (version 2).
https://www.viarail.ca/en/developer-resources

This application is free and open-source:
https://github.com/mtransitapps/ca-via-rail-train-android

This app is not related with the Via Rail Canada Inc.

Permissions:
- Other: required for real-time service statuses and to read news from viarail.ca and Twitter
