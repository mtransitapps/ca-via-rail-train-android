Cette application ajoute les informations des trains VIA Rail du Canada à MonTransit.

L'app contient les horaires planifiés ainsi que les nouvelles de viarail.ca et @VIA_Rail sur Twitter.

Les trains VIA Rail desservent Vancouver, Edmonton, Saskatoon, Winnipeg, Toronto, Ottawa, Montreal, Halifax... au Canada.

Une fois cette application installée, l'application MonTransit affichera les informations des trains (horaires ...).

Cette application a seulement une icône temporaire : télécharger l'app MonTransit (gratuit) dans la section "Autres ..." ci-dessous ou en cliquant sur ce lien Google Play https://bit.ly/MonTransitPlay

Vous pouvez installer cette application sur la carte SD mais ce n'est pas recommandé.

Ces horaires sont générées à partir du fichier GTFS fourni par VIA Rail sous Licence du gouvernement ouvert - Canada (version 2).
https://www.viarail.ca/fr/ressources-developpeurs

Cette application est gratuite et open-source : 
https://github.com/mtransitapps/ca-via-rail-train-android

Cette application n'est pas associée au Via Rail Canada Inc.

Autorisations :
- Autres : requis pour les états de service en temps réel et pour lire les nouvelles de viarail.ca et Twitter.
